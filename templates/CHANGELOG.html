How to add an entry ​

After saving a template via PowerBI Desktop:

  1. Note the bundle_schema_version from the manifest.yaml of the bundle you used during authoring.
  2. Open the saved template in a fresh Desktop session and verify all pages render with non-empty visuals.
  3. Append a row to the table above with format:
| 2026-05-09 | swao-report.pbit | 1.0 | 1.0.0 | OK -- 6 pages render against sovereign-health bundle |

Schema-bump procedure ​

When the star-schema columns change:

  • Patch (1.0.0 -> 1.0.1): additive, optional column. No template change needed; existing template still works. Append a row noting the bundle version.
  • Minor (1.0.0 -> 1.1.0): additive column that the template should display. Update the template (open, refresh, add visual, re-save). Bump template version to 1.1. Append a row.
  • Major (1.0.0 -> 2.0.0): column renamed or removed. Old template will fail to load. Rebuild against the new schema. Bump template version to 2.0. Append a row.
