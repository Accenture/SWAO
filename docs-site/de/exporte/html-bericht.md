# HTML-Bericht

SWAO erstellt nach jeder Bewertung einen eigenstandigen HTML-Bericht -- eine einzelne Datei
ohne externe Abhangigkeiten, geeignet fur Revisionsunterlagen und lesbar in jedem modernen Browser.

> Diese Seite ist auf Englisch vollstandig verfugbar: [HTML Report](/en/exports/html-report)

## Bericht erstellen

```bash
swao report --open
```

Vollstandige Dokumentation auf Englisch: [HTML Report](/en/exports/html-report)
