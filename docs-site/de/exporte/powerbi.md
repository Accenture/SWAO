# Power BI-Export

SWAO kann Bewertungsergebnisse in einem Format exportieren, das mit Microsoft Power BI Desktop
kompatibel ist -- fur Dashboards, Trendanalysen und Integration in bestehende
Unternehmensberichtsinfrastrukturen.

> Diese Seite ist auf Englisch vollstandig verfugbar: [Power BI Export](/en/exports/powerbi)

## Export ausfuhren

```bash
swao assess --app <name> --export powerbi
```

Vollstandige Dokumentation auf Englisch: [Power BI Export](/en/exports/powerbi)
