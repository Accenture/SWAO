# Terminal-Benutzeroberflache (TUI)

SWAO enthalt eine Vollbild-Terminal-Oberflache zur interaktiven Untersuchung von Bewertungsergebnissen.

> Diese Seite ist auf Englisch vollstandig verfugbar: [TUI Reference](/en/reference/tui)

## TUI starten

```bash
swao assess --app <name> --interactive
```

## Tastenbelegung

| Taste | Aktion |
|-------|--------|
| `Pfeiltasten` | Navigation zwischen Befunden |
| `Enter` | Befund aufklappen |
| `Ctrl+G` | Alle Befunde einklappen |
| `F` | Filterbereich offnen |
| `S` | Sortierreihenfolge wechseln |
| `E` | Aktuelle Ansicht als HTML exportieren |
| `Q` | TUI beenden |

Vollstandige Dokumentation auf Englisch: [TUI Reference](/en/reference/tui)
